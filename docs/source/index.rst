Welcome to CapitalCitiesGame's documentation!
=============================================

**CapitalCitiesGame**: provides a collection of python utilities that can be used to:

1. **learn** new capital cities, 
2. **play** with your friends.


If you find any issue with the software, please contact `me <mailto:daniele.traversa@unimi.it>`_, or report it  on `github <https://github.com/danitrave/CapitalCitiesGame/issues>`_.
